# j4img
javascript image manipulations (resize, rotate);

## Preview

![Preview](https://raw.githubusercontent.com/Corey-Maler/j4img/master/prev.gif)

## Usage
```javascript
var j4img = require('j4img');
j4img(img_DOM);
```

or:

```html
<script src="/js/j4img.js"></script>
<script>
   window.libs.j4img(img_DOM);
</script>
```
